# OreoFeatures
### **Project Goal :**
The main goal show the most important features of android oreo.

### **List of Samples contained on the Project**
1) Picture in Picture Feature
2) Notifications Dot and Notification Channel
3) Downloadable font
4) Firebase Job dispatcher
5) Job Scheduler
